## Практика 2 — try только с finally

```java
try {
    // ...
} finally {
    // ...
}
```

### Ответ

**Да, можно.** Это допустимая конструкция.

Блок `finally` выполнится всегда, а исключение (если возникнет) будет передано вверх по стеку вызовов.

```java
/**
 * Практика #2: try с finally без catch
 *
 * @author Arina
 */
public class TryFinallyExample {
    public static void main(String[] args) {
        try {
            method();
        } catch (Exception e) {
            System.out.println("Исключение поймано в main: " + e.getMessage());
        }
    }

    public static void method() {
        try {
            System.out.println("Внутри try");
            throw new RuntimeException("Ошибка");
        } finally {
            System.out.println("Finally выполнился");
        }
        // Исключение передаётся дальше
    }
}
```

### Вывод

```
Внутри try
Finally выполнился
Исключение поймано в main: Ошибка
```
