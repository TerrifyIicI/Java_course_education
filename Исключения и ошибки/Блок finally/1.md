## Практика 1 — try без catch и finally

```java
try {
    // ...
}
```

### Ответ

**Нет, нельзя.** Это вызовет ошибку компиляции.

Блок `try` обязательно должен сопровождаться хотя бы одним блоком `catch` или блоком `finally`.

```
error: 'try' without 'catch', 'finally' or resource declarations
```
