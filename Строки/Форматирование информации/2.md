## Практика 2 — Метод flush()

Метод `flush()` принудительно записывает все данные из внутреннего буфера в целевой поток (файл, сеть и т.д.).

### Зачем нужен

- Данные буферизуются для повышения производительности
- Без `flush()` данные могут оставаться в буфере и не записываться сразу
- При аварийном завершении программы буферизованные данные могут быть потеряны
- Важен при записи в файлы и сетевые потоки

### Пример

```java
import java.util.Formatter;
import java.io.FileWriter;
import java.io.IOException;

/**
 * Практика #2: Назначение метода flush()
 *
 * @author Arina
 */
public class FlushExample {
    public static void main(String[] args) {
        try {
            FileWriter fw = new FileWriter("output.txt");
            Formatter formatter = new Formatter(fw);
            
            formatter.format("Строка 1%n");
            formatter.format("Строка 2%n");
            
            // flush() принудительно записывает данные из буфера в файл
            formatter.flush();
            
            // Теперь данные гарантированно записаны в файл,
            // даже если программа завершится аварийно
            
            formatter.format("Строка 3%n");
            formatter.close();  // close() также вызывает flush()
            
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

**Примечание:** Метод `close()` автоматически вызывает `flush()` перед закрытием
