## Практика 3 — Три двойные кавычки (Text Blocks)

Три двойные кавычки (`"""`) используются для создания многострочных строковых литералов.

**В Java** текстовые блоки поддерживаются начиная с версии 15 (превью в Java 13-14).

```java
// Без текстовых блоков (старый способ)
String oldWay = "Line 1\n" +
                "Line 2\n" +
                "Line 3";

// С текстовыми блоками (Java 15+)
String textBlock = """
        Line 1
        Line 2
        Line 3
        """;
```

**Преимущества:**
- Не нужно экранировать кавычки и переносы строк
- Код более читаемый
- Удобно для JSON, HTML, SQL запросов

**Если используется версия Java ниже 15**, то для многострочных строк применяются:
- Символы переноса строки `\n`
- Конкатенация строк через `+`
- `StringBuilder`
- `String.format()`
