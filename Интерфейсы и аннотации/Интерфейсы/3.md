## Практика 3 — Protected интерфейс

```java
protected interface Interface {
    // ...
}
```

### Область видимости

**Ошибка компиляции** — интерфейс верхнего уровня не может быть объявлен как `protected`.

Спецификатор `protected` можно использовать только для вложенных интерфейсов:

```java
public class Outer {
    protected interface NestedInterface {
        void method();
    }
}
```

В этом случае `NestedInterface` доступен:
- Внутри класса `Outer`
- В классах того же пакета
- В подклассах `Outer` из других пакетов
