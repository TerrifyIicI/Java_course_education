## Практика 4 — Private интерфейс

```java
private interface Interface {
    // ...
}
```

### Область видимости

**Ошибка компиляции** — интерфейс верхнего уровня не может быть объявлен как `private`.

Спецификатор `private` можно использовать только для вложенных интерфейсов:

```java
public class Outer {
    private interface NestedInterface {
        void method();
    }
    
    // Используется только внутри Outer
    private class Inner implements NestedInterface {
        @Override
        public void method() {
            System.out.println("Implementation");
        }
    }
}
```

В этом случае `NestedInterface` доступен только внутри класса `Outer`
