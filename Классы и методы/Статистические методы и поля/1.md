Практика 1 — Варианты вызова статического метода
Код класса
```Java
public class A {
    public static int a = 1;
    public static int b;

    public static void printVars() {
        System.out.println(a);
        System.out.println(b);
    }
}
```
Все варианты вызова метода printVars()
```Java
/**
 * Практика #1: Варианты вызова статического метода
 *
 * @author Arina
 */
public class Main {
    public static void main(String[] args) {
        // 1. Вызов через имя класса (рекомендуемый способ)
        A.printVars();

        // 2. Вызов через экземпляр класса
        A obj = new A();
        obj.printVars();

        // 3. Вызов через переменную со значением null
        A nullObj = null;
        nullObj.printVars();  // Работает! NullPointerException не будет
    }
}
```
Пояснение
- Вызов через имя класса — рекомендуемый способ, так как явно показывает, что метод статический.
- Вызов через экземпляр — работает, но не рекомендуется, так как создаёт впечатление, что метод принадлежит объекту.
- Вызов через null-ссылку — работает, потому что статический метод привязан к классу, а не к объекту. Компилятор определяет тип переменной и вызывает метод класса, игнорируя значение ссылки.
