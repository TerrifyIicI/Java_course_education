## Практика 4 — Интерфейс AutoCloseable

Интерфейс `AutoCloseable` позволяет автоматически закрывать ресурсы при использовании конструкции **try-with-resources**. Это гарантирует освобождение ресурсов даже при возникновении исключения.

### Пример

```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

/**
 * Практика #4: Использование AutoCloseable с try-with-resources
 *
 * @author Arina
 */
public class Main {
    public static void main(String[] args) {
        String filePath = "example.txt";

        // BufferedReader реализует AutoCloseable,
        // поэтому автоматически закроется после выхода из блока try
        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.out.println("Ошибка при чтении файла: " + e.getMessage());
        }
        // reader.close() вызывается автоматически
    }
}
```

### Преимущества

- **Автоматическое закрытие** — не нужен блок `finally`
- **Меньше кода** — более чистый и читаемый код
- **Безопасность** — ресурс гарантированно закроется даже при исключении
