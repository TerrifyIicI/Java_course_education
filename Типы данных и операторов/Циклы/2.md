### Практика 2 — Операторы break и continue
Пример с `break`
```Java
/**
 * Цикл прерывается при нахождении числа 5
 *
 * @author Arina
 */
public class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                System.out.println("Найдено число 5, выходим из цикла");
                break;
            }
            System.out.println("i = " + i);
        }
    }
}
```
Вывод:
```text
i = 1
i = 2
i = 3
i = 4
Найдено число 5, выходим из цикла
```
Пример с `continue`
```Java
/**
 * Пропускаем чётные числа
 *
 * @author Arina
 */
public class ContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue;  // пропускаем чётные числа
            }
            System.out.println("Нечётное: " + i);
        }
    }
}
```
Вывод:
```text
Нечётное: 1
Нечётное: 3
Нечётное: 5
Нечётное: 7
Нечётное: 9
```
Пример с `break` в бесконечном цикле
```Java
/**
 *
 * @author Arina
 */
public class InfiniteLoopBreak {
    public static void main(String[] args) {
        int counter = 0;
        
        while (true) {
            counter++;
            System.out.println("Итерация: " + counter);
            
            if (counter >= 5) {
                System.out.println("Достигнут лимит, выходим");
                break;
            }
        }
    }
}
```
Вывод:
```text
Итерация: 1
Итерация: 2
Итерация: 3
Итерация: 4
Итерация: 5
Достигнут лимит, выходим
```
