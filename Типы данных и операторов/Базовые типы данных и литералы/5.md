```java
public class TypePromotion {
    public static void main(String[] args) {
        byte byteVal = 10;
        short shortVal = 20;
        int intVal = 30;
        long longVal = 40L;
        float floatVal = 1.5f;
        double doubleVal = 2.5;
        
        // byte + short = int
        int result1 = byteVal + shortVal;
        System.out.println("byte + short = " + result1 + " (тип int)");
        
        // int + long = long
        long result2 = intVal + longVal;
        System.out.println("int + long = " + result2 + " (тип long)");
        
        // long + float = float
        float result3 = longVal + floatVal;
        System.out.println("long + float = " + result3 + " (тип float)");
        
        // float + double = double
        double result4 = floatVal + doubleVal;
        System.out.println("float + double = " + result4 + " (тип double)");
        
        // int + double = double
        double result5 = intVal + doubleVal;
        System.out.println("int + double = " + result5 + " (тип double)");
    }
}
```
**Объяснение:** При выполнении арифметических операций с разными типами меньший тип автоматически расширяется до большего. Результат всегда имеет тип большего операнда.
