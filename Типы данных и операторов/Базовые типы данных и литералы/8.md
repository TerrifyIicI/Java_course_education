**Type Inference** — автоматическое определение типа переменной компилятором на основе присваиваемого значения.
```java
public class TypeInferenceExample {
    public static void main(String[] args) {
        // Вместо явного указания типа
        String message = "Hello";
        int number = 42;
        
        // Можно использовать var
        var messageVar = "Hello";  // компилятор выведет String
        var numberVar = 42;        // компилятор выведет int
        var piVar = 3.14;          // компилятор выведет double
    }
}
```
*Ограничения var:*
- Только для локальных переменных
- Переменная должна быть инициализирована при объявлении
- Нельзя инициализировать значением null
