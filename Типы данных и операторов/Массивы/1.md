## Практика 1 — Присваивание массивов

### Код

```java
int a1[] = {1,2,3,4,5};
int a2[] = {6,7,8,9,10};
a1 = a2;
```

### Что произойдёт

После выполнения `a1 = a2` переменная `a1` будет ссылаться на тот же массив, что и `a2`.

**До присваивания:**
- `a1` → `{1,2,3,4,5}`
- `a2` → `{6,7,8,9,10}`

**После присваивания:**
- `a1` → `{6,7,8,9,10}`
- `a2` → `{6,7,8,9,10}`

Обе переменные теперь указывают на один и тот же массив `{6,7,8,9,10}`. Исходный массив `{1,2,3,4,5}` больше не имеет ссылок и будет удалён сборщиком мусора. Изменение элементов через `a1` отразится на `a2` и наоборот:
```Java
a1[0] = 100;
System.out.println(a2[0]);  // Выведет: 100
```

```java
a1[0] = 100;
System.out.println(a2[0]);  // Выведет: 100
```
