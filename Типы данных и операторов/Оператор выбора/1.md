## Практика 1 — Допустимые типы в switch

В операторе `switch` можно использовать следующие типы:

- **byte, short, int, char** (и их классы-оболочки Byte, Short, Integer, Character)
- **String** (начиная с Java 7)
- **enum** (перечисления)

**Примечание:** Типы `long`, `float`, `double` и `boolean` в `switch` использовать **нельзя**.

### Примеры для всех допустимых типов

#### 1. int

```java
int intValue = 2;
switch (intValue) {
    case 1:
        System.out.println("один");
        break;
    case 2:
        System.out.println("два");
        break;
    default:
        System.out.println("другое");
        break;
}
```

#### 2. byte

```java
byte byteValue = 1;
switch (byteValue) {
    case 1:
        System.out.println("один");
        break;
    case 2:
        System.out.println("два");
        break;
}
```

#### 3. short

```java
short shortValue = 3;
switch (shortValue) {
    case 1:
        System.out.println("один");
        break;
    case 3:
        System.out.println("три");
        break;
}
```

#### 4. char

```java
char charValue = 'B';
switch (charValue) {
    case 'A':
        System.out.println("буква A");
        break;
    case 'B':
        System.out.println("буква B");
        break;
}
```

#### 5. String

```java
String stringValue = "hello";
switch (stringValue) {
    case "hello":
        System.out.println("привет");
        break;
    case "bye":
        System.out.println("пока");
        break;
}
```

#### 6. enum

```java
enum Day { MONDAY, TUESDAY, FRIDAY }

Day day = Day.MONDAY;
switch (day) {
    case MONDAY:
        System.out.println("понедельник");
        break;
    case FRIDAY:
        System.out.println("пятница");
        break;
}
```

#### 7. Integer (класс-оболочка)

```java
Integer integerValue = 5;
switch (integerValue) {
    case 5:
        System.out.println("пять");
        break;
    default:
        System.out.println("другое");
        break;
}
```
